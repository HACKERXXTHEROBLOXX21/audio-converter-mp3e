Audio Converter (.mp3e to .mp3)

This tool allows you to convert .mp3e and other audio files (.wav, .ogg, etc.) into .mp3.

How to Use

  1. Clone this repo or download the ZIP.
  2. Install Python and dependencies: 
    pip install pydub
# Also install FFmpeg
  3. Put your files in the audio_files folder
  4. Run the script: 
    python convert_audio.py

Support

Supports: .mp3e, .wav, .ogg, .flac, .aac, .m4a
